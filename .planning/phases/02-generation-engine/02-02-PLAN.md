<task type="auto">
  <name>Task 1: Implement generator module with CSV reading, agent creation, and batch processing</name>
  <files>src/chuckles_prime/generator.py</files>
  <action>
Create the core generation engine module with these functions:

**1. `read_input_titles(csv_path: str | Path) -> list[str]`**
- Read titles from a CSV file with a "title" column header
- Strip whitespace, skip empty rows
- Return list of title strings
- Raise FileNotFoundError if file missing, ValueError if no "title" column

**2. `create_agent(model, parody_tool, phone_tool) -> CodeAgent`**
- Create a smolagents CodeAgent configured for parody generation
- Parameters: model (OpenAICompatibleModel), parody_tool (loaded Tool), phone_tool (loaded Tool)
- Use `instructions=PARODY_INSTRUCTIONS` from prompts module (NOT system_prompt=)
- Set `additional_authorized_imports=["json"]`
- Set `return_full_result=True`
- Set `max_steps=15`
- Pass only `[phone_tool]` as tools to the agent (parody_tool is used outside the agent for pre-computation, the agent only needs phone_tool for verification)
- Return configured CodeAgent

**3. `_parse_agent_output(raw_output: str) -> list[ParodyCandidate]`** (private)
- Try `json.loads(raw_output)` first
- If JSON parse fails, try to extract JSON from the string (the agent sometimes wraps JSON in explanation text): search for first `{` and last `}`, try parsing that substring
- If still fails, return empty list (caller handles this as partial failure)
- From parsed JSON, extract "parody1" and "parody2" as ParodyCandidate objects
- Extract phonetic_scores and humor_note from "attempts" array if present (match attempt text to parody1/parody2)
- If only 1 parody found, return list with 1 candidate (not an error)
- If >2 parodies somehow, take first 2

**4. `_extract_trace(result) -> AgentTrace`** (private)
- Takes a RunResult object from agent.run()
- Extract steps from `result.steps` (list of dicts)
- Extract final_output from `str(result.output)`
- Extract token_usage: check if `result.token_usage` exists and has a `.dict()` method or can be converted to dict; if not, set to None
- Extract state from `result.state` (should be "success" or "max_steps_error")
- Return populated AgentTrace

**5. `generate_single(title, agent, parody_tool, config) -> GenerationRecord`**
- Pre-compute suggestions using `pre_compute_suggestions(title, config.funny_words, parody_tool)`
- Build prompt using `build_generation_prompt(title, suggestions, config.human_examples, config.preferences_text)`
- Run agent: `result = agent.run(task=prompt, reset=True)` -- reset=True clears agent memory between titles
- Parse output with `_parse_agent_output`
- Extract trace with `_extract_trace`
- Build and return GenerationRecord
- If agent returns max_steps_error state, still build the record but set error="max_steps_error: agent did not reach final_answer"
- Do NOT catch exceptions here -- let them propagate to generate_batch

**6. `generate_batch(titles, agent, parody_tool, config) -> list[GenerationRecord]`**
- Iterate over titles
- Call generate_single for each, wrapped in try/except
- On exception: create a GenerationRecord with error=str(e), empty candidates, and a minimal AgentTrace(steps=[], final_output="", token_usage=None, state="error")
- Print progress: `print(f"[{i+1}/{len(titles)}] {title} ... {'OK' if not record.error else 'ERROR: ' + record.error}")`
- Return list of all GenerationRecord objects (successes and failures)

IMPORTANT NOTES:
- Import CodeAgent from smolagents (NOT from smolagents.agents)
- Use `from chuckles_prime.types import ParodyCandidate, AgentTrace, GenerationRecord`
- Use `from chuckles_prime.tools import pre_compute_suggestions`
- Use `from chuckles_prime.prompts import PARODY_INSTRUCTIONS, build_generation_prompt`
- The agent gets only phone_tool (word_phonetic_analyzer) as a tool. The parody_tool (parody_word_suggester) is used outside the agent for pre-computation.
- For token_usage extraction: smolagents RunResult.token_usage may be a TokenUsage object with .input_tokens and .output_tokens attributes, or None. Convert to dict manually: `{"input_tokens": tu.input_tokens, "output_tokens": tu.output_tokens}` if available.
