**Step A: Create `src/chuckles_prime/csv_cleaner.py`**

Implement `clean_human_parodies(input_path, output_path) -> int` that:
1. Reads raw human_parodies.csv line-by-line (NOT via csv.reader for full file -- formats vary by zone)
2. Skips header row
3. For each non-empty line, detects format zone and extracts (input, output, explanation):

   - **Zone 1 (rows 2-13):** Clean 3-column CSV. Parse with csv.reader. Take columns as-is.
   - **Zone 2 (rows 14-24):** Markdown-numbered format like `13. **Chinchilla,Chintrilla**`. Regex: `^\d+\.\s*\*\*(.+?),\s*(.+?)\*\*$` on the first CSV field. Explanation is in second CSV field (strip quotes).
   - **Zone 3 (rows 25-68):** Slash-separated like `Chinchilla / Chintrilla: Detailed Explanation: The...` with explanation split across CSV columns. Regex: `^(.+?)\s*/\s*(.+?):\s*(?:Detailed\s+)?Explanation:\s*(.+)` on first field. Reassemble explanation from all remaining columns.
   - **Zone 4 (rows 69-~270):** Triple-quoted with `Parody: ""output""**` format. First field is `"""input"""`, second contains `Parody: ""output""**`, third is explanation. Strip triple quotes, extract parody with regex `Parody:\s*"*(.+?)"*\*{0,2}$`.
   - **Zone 5 (rows ~270-1234):** Clean 3-column CSV again (different source data). Parse with csv.reader. Take columns as-is.
   - **Fallback:** If line has 3+ CSV fields and doesn't match zones 2-4 patterns, treat as standard 3-column.

4. For each extracted row: clean encoding artifacts (`A(c)` -> `e`), strip leading/trailing whitespace and quotes, normalize whitespace in explanation text
5. Skip rows where input or output is empty after cleaning
6. Deduplicate by `(input.lower().strip(), output.lower().strip())`, keeping first occurrence
7. Write to output_path as clean CSV with `csv.DictWriter`, fieldnames=["input", "output", "explanation"], proper quoting
8. Return count of cleaned rows

**Step B: Create `src/chuckles_prime/config.py`**

Implement:

```python
from dataclasses import dataclass, field
from pathlib import Path
import json
import csv

@dataclass(frozen=True)
class AppConfig:
    """Immutable application configuration loaded from external files."""
    # LLM backend
    model_name: str
    api_base_url: str
    api_key_env_var: str

    # External data
    funny_words: dict[str, list[str]]
    preferences_text: str
    human_examples: list[tuple[str, str, str]]  # (input, output, explanation)

    # Paths (for reference)
    funny_words_path: Path
    preferences_path: Path
    human_examples_path: Path
```

Implement `load_config(settings_path: str | Path) -> AppConfig`:
1. Read settings JSON from `settings_path` (resolve path, expand `~`)
2. Load `funny_words_path` -> JSON dict (category -> word list)
3. Load `preferences_path` -> JSON, extract `"style_description"` key as `preferences_text` (or empty string if missing)
4. Load `human_examples_path` -> read cleaned CSV with `csv.DictReader`, collect as list of (input, output, explanation) tuples
5. Extract `model_name`, `api_base_url`, `api_key_env_var` from settings
6. Return frozen `AppConfig` with all fields populated
7. All paths resolved relative to settings file parent directory if they are relative paths. Use `Path.resolve()`.
8. Raise `FileNotFoundError` with clear message if any config file is missing
9. Raise `ValueError` if settings JSON is missing required keys

Settings JSON format:
```json
{
    "funny_words_path": "/path/to/funny_words.json",
    "preferences_path": "/path/to/preferences.json",
    "human_examples_path": "/path/to/human_examples_clean.csv",
    "model_name": "qwen-3-32b",
    "api_base_url": "https://api.cerebras.ai/v1",
    "api_key_env_var": "CEREBRAS_API_KEY"
}
```
